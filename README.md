# DYQ-Tools
自己编写的实用小工具。

## ansible
Ansible自动部署

## tdit
巡检脚本

## tdmanager
python编写的集群管理工具。

## econvert.sh
错误代码转换
使用方法：
./econvert.sh -217363882

## ipconvert.sh
IP地址转换
使用方法：
./ipconvert.sh  0x2a90123

## vcalc.sh
统计vnode在不同节点的分布.
使用方法：
./valc.sh

## wal.pl
用来查看mnode的wal日志
使用方法：
perl wal.pl < {dataDir}/dnode/mnode/wal/wal0
