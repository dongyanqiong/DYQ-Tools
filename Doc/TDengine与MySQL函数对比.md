# TDengine 与 MySQL 函数对比

以下对比选用的版本如下：
- TDengine 3.1.0.3
- MariaDB 5.5.68

## 数学函数

### CEIL

**功能说明**: 向上取整

**差异**：
- MySQL 返回整数（但并非int类型）；
- TDengine 返回类型与原始类型一致

```sql
MariaDB [test]> select ceil(0.22);
+------------+
| ceil(0.22) |
+------------+
|          1 |
+------------+
1 row in set (0.00 sec)

MariaDB [test]> select ceil(1234567890123456789012345678901234567890.22);
+---------------------------------------------------+
| ceil(1234567890123456789012345678901234567890.22) |
+---------------------------------------------------+
|          1234567890123456789012345678901234567891 |
+---------------------------------------------------+
1 row in set (0.00 sec)
```

```sql
taos> select ceil(0.22) from tt;
        ceil(0.22)         |
============================
         1.000000000000000 |

taos> select ceil(1234567890123456789012345678901234567890.22) from tt;
 ceil(1234567890123456789012345678901234567890.22) |
====================================================
                             1.234567890123457e+39 |
```

### FLOOR

**功能说明**: 向下取整

**差异**：
- MySQL 返回整数（但并非int类型）；
- TDengine 返回类型与原始类型一致


```sql
MariaDB [test]> select floor(1.22);
+-------------+
| floor(1.22) |
+-------------+
|           1 |
+-------------+
1 row in set (0.00 sec)

MariaDB [test]> select floor(1234567890123456789012345678901234567890.22);
+----------------------------------------------------+
| floor(1234567890123456789012345678901234567890.22) |
+----------------------------------------------------+
|           1234567890123456789012345678901234567890 |
+----------------------------------------------------+
1 row in set (0.00 sec)
```

```sql
taos> select floor(1.22) from tt;
        floor(1.22)        |
============================
         1.000000000000000 |
Query OK, 1 row(s) in set (0.001872s)

taos> select floor(1234567890123456789012345678901234567890.22) from tt;
 floor(1234567890123456789012345678901234567890.22) |
=====================================================
                              1.234567890123457e+39 |
Query OK, 1 row(s) in set (0.001250s)
```
